WebGIS Google Map API (Point, Line, Polygon)

DESKRIPSI SINGKAT

WebGIS disamping dibuat dengan menggunakan Google Map API. Adapun data yang ditampilkan meliputi tiga macam, yakni berupa titik (point), garis (polyline), dan area (polygon). Animasi bounce dan info window ditambahkan supaya lebih menarik.
